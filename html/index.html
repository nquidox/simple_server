Simple server

Paths:

  • "/" - This page. Handles GET and POST requests, other methods not allowed. Also gives you a cookie.
  • "/mux" - just a test function, nothing interesting.
  • "/temp" - returns fake temperature value in text format.
  • "/json" -returns fake temperature, humidity and pressure values in JSON format.
  • "/params" - returns a string of URL path except domain.
  • "/middleware" - returns message from simple middleware function.
